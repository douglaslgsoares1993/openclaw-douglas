# Politica inicial de uso por colegas

- Nao inserir dados sigilosos sem autorizacao.
- Nao inserir senhas, tokens ou credenciais.
- Revisar respostas antes de qualquer uso oficial.
- IA auxilia, nao decide.
- Preservar responsabilidade funcional.
- Separar uso administrativo e investigativo.
- Registrar limitacoes conhecidas.
- Usar linguagem institucional e conferir fatos.
- Reportar erro, resposta inadequada ou vazamento de contexto.
