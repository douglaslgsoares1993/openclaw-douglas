# Concurso Delegado de Polícia

Skill customizada para preparação ao concurso de Delegado de Polícia.

## Quando usar
- Usuário envia questão de concurso para resolver
- Usuário pede para simular prova objetiva
- Usuário pede análise de peça discursiva
- Usuário pede simulação de prova oral
- Usuário pede pesquisa de jurisprudência para concurso

## Foco principal
- Banca: CESPE/Cebraspe
- Prioridade: PCPE, estados do Nordeste, Distrito Federal
- Pontos fortes de Douglas: Direito Penal, Processo Penal, Criminologia
- Pontos de atenção: Direito Constitucional, Medicina Legal

## Como resolver questões CESPE
1. Identifica o tema central e o enunciado (certo/errado ou múltipla escolha)
2. Apresenta a resposta direta
3. Explica o fundamento legal/doutrinário/jurisprudencial
4. Aponta o erro ou acerto da assertiva com precisão
5. Menciona divergências relevantes se existirem

## Como analisar peças discursivas
1. Verifica estrutura (introdução, desenvolvimento, conclusão)
2. Avalia fundamentação jurídica
3. Verifica linguagem técnica
4. Pontua como examinador CESPE pontuaria
5. Sugere melhorias objetivas

## Jurisprudência prioritária
- STF: controle de constitucionalidade, direitos fundamentais, processo penal constitucional
- STJ: direito penal, processo penal, habeas corpus
- TJPE: decisões relevantes para a prática policial em PE
